## Overview

5 tools for fetching Google Ads data — campaigns, daily trends, geographic and device breakdowns, and search terms.

<Note>
  Google Ads data has a **~1-day delay**. Use `today - 1 day` as the most recent `end_date`.
</Note>

---

## gads_get_campaigns

Get Google Ads campaign performance data.

**Parameters:**

| Parameter | Type | Required | Default | Description |
|-----------|------|----------|---------|-------------|
| `project_id` | string | Yes | — | Project ID from `list_projects` |
| `start_date` | string | Yes | — | Start date (YYYY-MM-DD) |
| `end_date` | string | Yes | — | End date (YYYY-MM-DD) |
| `campaign_type` | string | No | All | Filter: `"SEARCH"`, `"PERFORMANCE_MAX"`, `"DISPLAY"`, `"VIDEO"`, `"SHOPPING"` |
| `sort_by` | string | No | `cost` | Sort: `"cost"`, `"conversions"`, `"roas"`, `"impressions"`, `"clicks"` |
| `limit` | int | No | 50 | Max results (max 100) |

**Response:**

```json
{
  "_meta": { "total": 12, "returned": 12 },
  "currency": "TRY",
  "items": [
    {
      "id": "123456789",
      "name": "Brand - Search",
      "type": "SEARCH",
      "status": "ENABLED",
      "budget_amount": 500.0,
      "impressions": 25000,
      "clicks": 3500,
      "cost": 1200.50,
      "ctr": 14.0,
      "cpc": 0.34,
      "conversions": 180,
      "conversion_value": 9500.0,
      "conversion_rate": 5.14,
      "cpa": 6.67,
      "roas": 7.91
    }
  ]
}
```

**Example Prompt:**

> "Show me my best-performing campaigns sorted by ROAS"

## gads_get_daily_metrics

Get daily Google Ads metrics — impressions, clicks, cost, conversions as a time series.

**Parameters:**

| Parameter | Type | Required | Default | Description |
|-----------|------|----------|---------|-------------|
| `project_id` | string | Yes | — | Project ID from `list_projects` |
| `start_date` | string | Yes | — | Start date (YYYY-MM-DD) |
| `end_date` | string | Yes | — | End date (YYYY-MM-DD) |
| `campaign_id` | string | No | All | Specific campaign ID, or empty for all campaigns |

**Response:**

```json
{
  "_meta": { "total": 30, "returned": 30 },
  "currency": "TRY",
  "items": [
    {
      "date": "2026-06-01",
      "impressions": 850,
      "clicks": 120,
      "cost": 42.50,
      "conversions": 6,
      "conversion_value": 320.0,
      "ctr": 14.1,
      "cpc": 0.35,
      "roas": 7.53,
      "cpa": 7.08
    }
  ]
}
```

**Example Prompt:**

> "Show me daily ad spend trends for the last 30 days"

## gads_get_search_terms

Get actual search terms that triggered your Google Ads (Search campaigns only).

**Parameters:**

| Parameter | Type | Required | Default | Description |
|-----------|------|----------|---------|-------------|
| `project_id` | string | Yes | — | Project ID from `list_projects` |
| `start_date` | string | Yes | — | Start date (YYYY-MM-DD) |
| `end_date` | string | Yes | — | End date (YYYY-MM-DD) |
| `campaign_id` | string | No | All Search | Specific Search campaign ID, or empty for all |
| `limit` | int | No | 50 | Max results (max 100) |

**Response:**

```json
{
  "_meta": { "total": 250, "returned": 50 },
  "currency": "TRY",
  "items": [
    {
      "search_term": "seo aracı",
      "status": "ADDED",
      "impressions": 500,
      "clicks": 75,
      "cost": 25.0,
      "conversions": 4,
      "conversion_value": 200.0,
      "ctr": 15.0,
      "cpc": 0.33,
      "conversion_rate": 5.33,
      "roas": 8.0,
      "cpa": 6.25
    }
  ]
}
```

<Warning>
  Only available for **Search campaigns** — not Performance Max, Display, or Video campaigns.
</Warning>

<Tip>
  Look for high-cost, zero-conversion terms and add them as negative keywords. Compare with organic keywords (`get_keywords`) to find paid/organic overlap.
</Tip>

**Example Prompt:**

> "What search terms are triggering my ads? Find any wasted spend"
